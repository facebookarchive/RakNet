Project: Autopatcher Client Restarter

Description: The autopatcher client sometimes requires a restart of the application. It outputs a text file with instructions on what to do (copy files, etc). This application parses that text file and should be included with an application that uses the autopatcher client.

Dependencies: None

Related projects: AutopatcherClientRestarter, AutopatcherPostgreSQLRepository, AutopatcherServer
