Project: RakVoice

Description: Tests / demonstrates RakVoice. This project uses PortAudio to record and play back sound. RakVoice itself is a plugin which does not depend on PortAudio

Dependencies: RakVoice has no dependencies. However, this sample depends on PortAudio. PortAudio is included with the distribution under DependentExtensions\portaudio_v18_1

Related projects: RakVoiceFMOD
