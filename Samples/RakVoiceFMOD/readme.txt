Project: RakVoiceFMOD

Description: Tests / demonstrates RakVoice. This project uses FMOD to record and play back sound. RakVoice itself is a plugin which does not depend on PortAudio

Dependencies: RakVoice has no dependencies. However, this sample depends on FMOD. FMOD is not included with the distribution. 
The sample assumes FMOD is installed at C:\Program Files\FMOD SoundSystem\FMOD Programmers API Win32\api\inc
If this is not the case for you, change the path accordingly.
